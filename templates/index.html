🧠 AI Model
TensorFlow deep learning — 3-class classifier (Normal / Probe / DoS)
🎯 Attack Classification Reference
Label 0
✅ Normal Traffic
Regular network activity with no suspicious patterns. Low error and warning counts.
Label 1
⚠️ Probe / Scan Attack
Reconnaissance activity — port scans, vulnerability probing, or unauthorized discovery attempts.
Label 2
🚨 DoS / DDoS Attack
Denial of Service attack — high volume traffic designed to overwhelm or crash systems.
